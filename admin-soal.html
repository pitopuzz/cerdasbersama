🔧 Sync Akses User dari Order

Tool ini membaca semua order berstatus paid lalu mengupdate field subBabAkses di setiap user sesuai sub-bab yang dibeli. Gunakan ini untuk memperbaiki user yang sudah bayar tapi belum bisa akses latihan.

⚠️ Proses ini aman — hanya menambah akses, tidak menghapus akses yang sudah ada.
📖 Cara Tambah & Update Soal
Cara 1 — Tambah Manual (1 soal)
1

Isi form di tab "Tambah Manual"

Pilih mapel, tulis sub-bab, soal, 5 pilihan jawaban, dan pembahasan.

2

Pilih kunci jawaban

Klik tombol A/B/C/D/E — tombol akan jadi merah saat dipilih.

3

Klik Simpan

Soal langsung masuk Firestore dan bisa dimainkan di halaman latihan.

Cara 2 — Bulk Upload Excel (banyak soal)
1

Download template Excel

Di tab "Bulk Excel" → klik Download Template. Buka di Excel/Google Sheets.

2

Isi soal di Excel

Setiap baris = 1 soal. Isi semua kolom: kategori, subKategori, pertanyaan, opsiA–E, jawaban (tulis A/B/C/D/E), pembahasan, tingkat (mudah/sedang/sulit), akses (gratis/premium).

3

Simpan sebagai .xlsx

File → Save As → pilih format Excel Workbook (.xlsx).

4

Upload & preview

Di tab "Bulk Excel" → upload file → cek preview tabel → pastikan semua soal valid (tidak ada kolom kosong).

5

Pilih mode upload

Hapus Soal Lama + Upload Baru = reset semua soal lama dulu, cocok untuk update total.
Tambah ke Soal yang Ada = soal baru ditambahkan, soal lama tetap.

💡 Tips: Nama sub-bab harus konsisten — "Bilangan Bulat" ≠ "bilangan bulat". Pastikan kolom jawaban diisi A/B/C/D/E (huruf kapital).
